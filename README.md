# 🍣 Nagomi 和み

Nagomi é uma aplicação web desenvolvida com **React.js**, inspirada na cultura e gastronomia japonesa. O projeto foi criado com o objetivo de aprimorar minhas habilidades em React, explorando conceitos modernos de desenvolvimento web, componentização e criação de interfaces responsivas.

## 🚀 Demonstração

Acesse o projeto online:

🔗 https://sophiaoamaral.github.io/Nagomi/

## 📖 Sobre o Projeto

O Nagomi busca proporcionar uma experiência visual agradável e elegante, transmitindo a identidade de um restaurante japonês através de seu design e navegação.

Durante o desenvolvimento, foram aplicados conceitos como:

* Componentização com React
* Estruturação e organização de código
* Responsividade para diferentes dispositivos
* Animações e transições com CSS
* Boas práticas de desenvolvimento Front-End
* Reutilização de componentes

## 🛠️ Tecnologias Utilizadas

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Git & GitHub

## 📂 Instalação

Clone o repositório:

```bash
git clone https://github.com/SophiAoAmaral/Nagomi.git
```

Acesse a pasta do projeto:

```bash
cd Nagomi
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Este projeto foi desenvolvido como parte do meu processo de aprendizado e evolução com React, permitindo colocar em prática conceitos fundamentais para a construção de aplicações modernas e escaláveis.
